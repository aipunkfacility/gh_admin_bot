# Changelog (История версий)

Все детальные логи изменений хранятся в папке [history](./history/).

## [v3.0.0] - 2026-01-29

### Added

- **E2E Testing Layer**: Интегрирован Playwright, настроен Chromium и создан первый тест воронки бронирования (`1 passed`).
- **DevOps Automation**: Установлен Husky и настроен pre-commit хук для автоматической валидации документации через `verify_docs.js`.
- **SEO Automation v1.0**:
  - Создан выделенный навык `seo-generator` для ИИ.
  - Реализовано управление через `scripts/seo_tool.js` (Export/Import).
  - Внедрен **JSON-LD (Schema.org)** на лендинг для индексации всех категорий (27+ позиций).
  - Капитальный рефакторинг Admin UI: метаданные вынесены в отдельную карточку.
  - **Google Search Console**: Пройдена верификация (HTML-файл + Meta-tag).
  - **Google Analytics 4**: Подключен сбор данных (ID: `G-86N4BEJ2KS`).
- **Bugfixes**:
  - Исправлен роутинг в админке для раздела "Настройки сайта" (теперь корректно подгружает `main`).
  - Исправлены ошибки деплоя, связанные с парсингом команд в PowerShell.

## [v2.3.0] - 2026-01-28

### Added

- Полная миграция на Supabase.
- Новая система управления администраторами (Hybrid).
- Детальная документация логики (`BOT_LOGIC.md`, `ADMIN_LOGIC.md`, `SITE_LOGIC.md`).
- Реорганизация файловой структуры: созданы папки `docs/`, `infra/`, `reports/`. Корень проекта очищен.
- Автоматизация дисциплины: внедрено правило "Human-in-the-loop" и навыки синхронизации памяти.

### Fixed

- Аварийное восстановление сервера (502 Gateway).
- Ошибки сохранения мета-данных сайта.
- Смещение Hero-блока на фронтенде.
